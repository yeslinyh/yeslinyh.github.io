Z-Library 账号注册教程

不用去网页注册

直接发邮箱即可.

收件人

blackbox@1delivery.re
标题和内容随意写,什么都可以.直接发送邮件即可.
可尝试使用下面的邮箱来发送邮件
https://wx.mail.qq.com/
https://mail.163.com/
https://www.google.com/intl/zh-CN/gmail/about/

发送一封邮件之后,可能要等15-20分钟.
就会收到对面回复的邮件.
如果20分钟后,还是没看见回复邮件,打开[垃圾箱]看看.

登录账号就是你的邮箱
密码在回复的邮件里,会告诉你密码.
